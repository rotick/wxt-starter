# WXT + Vue 3

This template should help get you started developing with Vue 3 in WXT.

```bash
git clone --depth=1 https://github.com/rotick/wxt-starter.git new-project

cd new-project

rm -rf .git
mv .env.example .env
```

Replace all:

- wxt-starter -> { your extension name }

## Development

```bash
# install
pnpm install

# development
pnpm run dev
```
